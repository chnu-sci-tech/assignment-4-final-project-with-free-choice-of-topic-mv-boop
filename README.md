# IDA Lab 4: Titanic Exploratory Data Analysis (EDA)

## Опис проекту
Цей проект виконано в рамках Лабораторної роботи №4.
**Мета роботи:** Проведення розвідувального аналізу даних (EDA) на основі датасету пасажирів Титаніка для виявлення факторів, що впливали на ймовірність виживання.

Тема обрана зі списку рекомендованих: *Exploratory Data Analysis (EDA) and Visualization*.

## Технології
Проект реалізовано мовою **Python** у середовищі Jupyter Notebook. Використані бібліотеки:
* **Pandas:** для завантаження, очищення та маніпуляцій з даними.
* **Seaborn & Matplotlib:** для візуалізації результатів.
* **Numpy:** для математичних операцій.

## Основні етапи роботи
1.  **Збір даних:** Використано класичний датасет Titanic (локальний файл `titanic.csv`).
2.  **Очищення даних (Data Cleaning):**
    * Заповнено пропуски у віці (`age`) медіанним значенням.
    * Видалено колонку `deck` через критичну кількість пропусків (>75%).
    * Видалено незначну кількість рядків без даних про порт посадки (`embarked`).
3.  **Візуалізація:** Побудовано графіки залежностей виживання від статі, класу квитка та віку.

## Ключові висновки (Findings)
Аналіз даних показав наступні закономірності:
* **Стать:** Жінки мали значно вищі шанси на виживання (~74%) порівняно з чоловіками (~19%).
* **Соціальний статус:** Пасажири 1-го класу виживали найчастіше (>60%), тоді як пасажири 3-го класу мали найменші шанси (<25%).
* **Вік:** Діти (0-10 років) мали пріоритет при порятунку.
* **Кореляція:** Існує сильний зворотний зв'язок між класом квитка та виживанням (чим кращий клас, тим вищі шанси).
